

## 简介

一直想用node相关的框架做一款前后端分离的后台管理系统，所以抽空写了一份基础版。主要使用Egg.js + ts + mysql,加密使用utility做的MD5（可以更换成别的加密形式，比如base64加盐），令牌是jsonwebtoken。目前只是完成一个雏形，只是完成登录与用户列表与新增。前端工程目前准备用Vue+Antd写一份，有空在React+Antd写一份
